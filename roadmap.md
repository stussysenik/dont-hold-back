# Roadmap

> Where this goes next. Slowly, deliberately — like the experience itself.

<br />

## Now — Polish the Ritual

The core is built. These are the refinements that make it feel *inevitable*.

- [ ] **Tune the singing bowl** — experiment with frequencies, add subtle detuning for warmth
- [ ] **Ink bleed refinement** — smoother noise masking, more calligraphic feel
- [ ] **Mobile testing** — verify touch hold on iOS Safari + Android Chrome
- [ ] **Haptic feedback** — subtle vibration on mobile when the hold completes
- [ ] **Preloader** — elegant font loading gate (no flash of unstyled text)

<br />

## Next — Deepen the Experience

- [ ] **Dark/light duality** — a secret light mode (cream background, dark text) toggled by a gesture
- [ ] **Custom text** — let users type their own mantra while keeping the same ritual
- [ ] **Seasonal palettes** — warm amber in autumn, cool blue in winter (based on date, not random)
- [ ] **Shareable links** — `dontholdback.app/your-words-here`
- [ ] **PWA / offline** — service worker so it works without internet, like a real bookmark

<br />

## Later — Expand the Canvas

- [ ] **MP4 export** — FFmpeg.wasm for universal video format
- [ ] **Instagram Stories integration** — direct share via Web Share API
- [ ] **Wallpaper mode** — generate static wallpapers at any resolution
- [ ] **Print export** — high-DPI PDF for physical prints
- [ ] **Sound customization** — choose between singing bowl, rain, or silence

<br />

## Someday — Beyond the Screen

- [ ] **Physical edition** — e-ink display that shows the breathing text forever
- [ ] **API** — embed the ritual in other sites as a component
- [ ] **Gallery** — community-submitted mantras, each with the same ritual treatment
- [ ] **Multi-language** — the same experience in any language, any script

<br />

---

*The roadmap is a direction, not a deadline. This project moves at the speed of intention.*
